Quick Start Guide

1

Analyze Historical Data

Start with the Prior Events Analysis to understand patterns from similar past events and establish baseline expectations.

2

Create Demand Inputs

Use the Demand Inputs tool to model transportation demand for your upcoming event with custom parameters.

3

Plan Transportation Management

Develop Supply inputs using the Supply Inputs tool to optimize traffic flow and minimize congestion.

4

Evaluate Performance

After your event, use Post Event Analysis to evaluate actual performance against predictions and update future scenarios.
